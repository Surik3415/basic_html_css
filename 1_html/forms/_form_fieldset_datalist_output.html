Елемент fieldset використовується для групування пов’язаних даних у формі.

Елемент legend визначає заголовок для fieldset елемента.

Personalia:




Елемент datalist визначає список попередньо визначених параметрів для input елемента. Під час введення даних користувачі бачитимуть розкривний список із попередньо визначеними параметрами. Атрибут listелемента input має посилатися на idатрибут елемента datalist

Елемент output представляє результат обчислення (як обчислення, виконаного сценарієм).
